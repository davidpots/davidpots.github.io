About

Since 2005, I've worked professionally to combine my UX/UI design and front-end development skills toward non-profit, enterprise, and consumer web projects. Currently, I'm a proud member of the UX team at LivingSocial, where I help design products and interactions for screens of all sizes. Outside of work, I'm a proud family man who enjoys podcasts and loves playing the guitar.

You may want to follow me on Twitter or Dribbble or GitHub or LinkedIn.

Featured Project

Songnotes

I designed & developed Songnotes to help me share my growing collection of curated guitar tabs, chords, lyrics, and lessons with aspiring musicians across the world. To do this, I built a Rails-powered CMS that feeds my hand-crafted song metadata snippets into a carefully designed framework that was built to support the structure of any song. The result is high-quality, consistent, and media-rich music notation that works for today's web-native audience.

Other Projects

Wisp

Wisp is an ultra-lightweight HTML framework to help you quickstart any project or rapid prototype. Quite simply, it sets up a generic project directory with an index.html file already linked to the usual CSS and jQuery resources. For simple projects where Rails or the HTML5 Boilerplate is overkill, Wisp helps you hit the ground running without any overhead.

Firefly Grove

Firefly Grove is a personal songwriting project showcasing 9 lo-fi folk songs I wrote & recorded dating back to 2008-2009. From a technical viewpoint, I used this project as an opportunity to introduce myself with the Soundcloud API & SDK while also exploring some jQuery layout techniques.
